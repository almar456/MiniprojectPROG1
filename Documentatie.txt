Stallen.py
Aanroepen: stallen(fietscode(als int of str))

Returnwaarden:
return 0 -> Het programma is voltooid, de fiets is gestald en op True gezet in de csv-file.
return 1 -> Het ingevoerde fietsnummer is incorrect.
return 2 -> De fiets staat al gestald in de stalling.

Functie:
Zet 'gestald' op True bij een ingevoerd fietsnummer, tenzij deze al op True staat.


Ophalen.py
Aanroepen: ophalen(username(als string), wachtwoord(als string))

Returnwaarden:
return 0 -> Het programma is voltooid, de fiets is uit de stalling en staat op False in de csv-file.
return 1 -> De ingevoerde combinatie van username en wachtwoord is incorrect.
return 2 -> De fiets is niet gestald in de stalling.

Functie:
Zet 'gestald' op False als de ingevoerde username en wachtwoord correct zijn, tenzij 'gestald' al op False staat.

Registreren.py:
Aanroepen: registereren(wachtwoord(als string), username(als string), volledigenaam(als string))

Returnwaarden:
return 0 -> Het programma heeft de gebruiker geregistreerd. De gebruiker wordt nu toegevoegd aan de csv-file.
return 1 -> De gebruiker heeft een wachtwoord ingevoerd dat niet aan de requirementst voldoet.

Functie:
Registreert de gebruiker en voegt de ingevoerde gegevens toe aan de csv-file. De gebruiker krijgt ook een 4 cijferige code.
